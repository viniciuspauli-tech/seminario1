Display: block

Display: block O elemento ocupa toda a largura disponível da tela e sempre começa em uma nova linha. Mesmo que tenha espaço ao lado, ele “quebra” a linha automaticamente. Permite definir largura, altura, margem e padding normalmente.

Display: grid O grid é usado para criar layouts, organizando elementos em linhas e colunas estruturadas. O grid facilita a criação de layouts complexos e responsivos, permitindo posicionar itens precisamente sem depender apenas de margens e espaçamentos.

Inline: Grind

Inline: grind Controla como o elemento é posicionado na página. Pode ser estático (padrão), relativo, absoluto, fixo ou sticky, alterando como ele se comporta em relação ao fluxo normal do documento.

flow-root

flow-root Impede o Colapso de Margens: Ele evita que a margem de um elemento filho "escape" e se junte com a margem do elemento pai (margin collapse).

tablet

Tablet Cria tabelas, muitos preferem usar apenas table mas no geral os dois servem para a criação de tabelas podendo organizar suas informações de forma que elas não fiquem confusas.

Display: inline-flow

Display: inline-block Inline flow é a forma moderna e explícita de definir o que conhecemos simplesmente como display: inline.

Display: inline-flex

Display: inline-block inline flex é o equivalente ao inline-block, mas com os superpoderes do Flexbox para alinhar o que está dentro dele.

Display: block-grid

Display: inline-block Inline grid é o meio-termo perfeito quando você quer o poder das colunas e linhas do Grid, mas não quer que o contêiner ocupe a largura toda da página.

Display: block-flow-root

Display: inline-block inline flow-root é a combinação perfeita entre um elemento que flui no texto e um contêiner que aceita dimensões fixas.

display: block flex

display: block flex combina as características de block e flex, permitindo que um elemento se comporte como um bloco e, ao mesmo tempo, utilize o modelo de layout flexível para seus filhos.

display: block grid

display: block grid Semelhante ao flex, o valor display: block grid é a forma moderna e explícita de escrever o que conhecemos simplesmente como display: grid

Display: block

Display: block O elemento ocupa toda a largura disponível da tela e sempre começa em uma nova linha. Mesmo que tenha espaço ao lado, ele “quebra” a linha automaticamente. Permite definir largura, altura, margem e padding normalmente.

Display: inline

Display: inline Inline no CSS faz com que elementos HTML se mantenham na mesma linha, ocupando apenas a largura necessária para o elemento, sem iniciar uma nova linha.

Display: inline-flex

Display: flex Inline-flex no CSS transforma um elemento em um contêiner flexível, que se comporta como um elemento inline em relação aos seus elementos vizinhos.

Display: grid

Display: grid O grid é usado para criar layouts, organizando elementos em linhas e colunas estruturadas. O grid facilita a criação de layouts complexos e responsivos, permitindo posicionar itens precisamente sem depender apenas de margens e espaçamentos.

Position Controla como o elemento é posicionado na página. Pode ser estático (padrão), relativo, absoluto, fixo ou sticky, alterando como ele se comporta em relação ao fluxo normal do documento.

Position

Position: relative O elemento continua ocupando seu espaço normal na página, mas pode ser deslocado usando top, bottom, left ou right. Também serve como referência para elementos com position: absolute dentro dele.

Position: absolute O elemento sai do fluxo normal da página e é posicionado em relação ao elemento pai que tenha position diferente de static. Pode ser colocado exatamente onde você quiser usando top, right, bottom e left.

Position: Static

Position: Static O elemento segue o fluxo normal do documento, as propriedades TOP, RIGHT, BOTTOM e LEFT não tem efeito.

Position: Relative

Position: Relative O elemento permanece no fluxo normal, mas pode ser deslocado em relação a sua posição original usando TOP, LEFT, etc. O espaço original dele é preservado.

Position: Absolute

Position: Absolute O elemento é removido do fluxo normal e posicionado em relação ao ancestral mais próximo com position ≠ static. Se não houver nenhum, usa o .

Position: Fixed

Position: Fixed O elemento é removido do fluxo e posicionado em relação à viewport (janela do navegador). Ele não rola com a página — fica fixo na tela.

Position: sticky

Position: sticky Híbrido entre relative e fixed. O elemento se comporta como relative até atingir um limite de rolagem definido, momento em que "gruda" como fixed.

Display: block

Display: block O elemento ocupa toda a largura disponível da tela e sempre começa em uma nova linha. Mesmo que tenha espaço ao lado, ele “quebra” a linha automaticamente. Permite definir largura, altura, margem e padding normalmente. Elementos ocupam toda a largura disponível e sempre começam em uma nova linha.

Display: inline Inline no CSS faz com que elementos HTML se mantenham na mesma linha, ocupando apenas a largura necessária para o elemento, sem iniciar uma nova linha. Elementos ficam na mesma linha e não aceitam largura/altura definidas.

Display: inline-grid

Display: inline-grid Inline grid é o meio-termo perfeito quando você quer o poder das colunas e linhas do Grid, mas não quer que o contêiner ocupe a largura toda da página.

Display: block-flow-root

Display: block-flow-root block-flow-root é a combinação perfeita entre um elemento que flui no texto e um contêiner que aceita dimensões fixas.

display: block flex

display: block flex combina as características de block e flex, permitindo que um elemento se comporte como um bloco e, ao mesmo tempo, utilize o modelo de layout flexível para seus filhos.

display: block grid

display: block grid Semelhante ao flex, o valor display: block grid é a forma moderna e explícita de escrever o que conhecemos simplesmente como display: grid

display: inline flow-root

display inline flow-root: Quando você quer manter o fluxo de texto e elementos clássicos (como parágrafos e spans), mas quer que as margens respeitem os limites da caixa.

display: inline flow

displayinline flow: Inline flow é a forma moderna e explícita de definir o que conhecemos simplesmente como display: inline;.
